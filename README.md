# Out of the Cave

A terminal text adventure where you play as Plato, leading cave-dwellers to build civilization.

## Requirements

- Python 3.11+

## Getting Started

```bash
# Install the project
pip install -e .

# Run the game
cave
```

Or run directly without installing:

```bash
python -m src.main
```

## Optional Dependencies

```bash
# AI features
pip install -e ".[ai]"

# Development/testing
pip install -e ".[dev]"
```
